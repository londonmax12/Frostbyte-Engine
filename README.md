# Frostbyte
Frostbyte is a game engine written in C++
## Generating project files
### Windows
#### Prerequisites
- git
- cmake (>=3.6)
- Vulkan SDK
#### Building projects
```
git clone https://github.com/londonmax12/Frostbyte-Engine
cd Frostbyte-Engine/scripts
setup.bat
```
### Linux
#### Prerequisites
- git
- cmake (>=3.6)
- make
- Vulkan SDK
#### Building projects
```
git clone https://github.com/londonmax12/Frostbyte-Engine
cd Frostbyte-Engine/scripts
./setup.sh
```
